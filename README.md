# Prabesh Shrestha — Portfolio

Personal portfolio site built for COMP 3078 (Capstone Project II) at George Brown College.

## What's in here

A single-page portfolio covering my background, work experience, capstone project (Shiftora), academic work samples, certifications, and contact info.

**Stack:** Plain HTML, CSS, and vanilla JS — no frameworks, no build step. Just open `index.html` in a browser.

## Files

```
index.html    — all the content
styles.css    — all styling
script.js     — scroll animations, nav highlight, cursor glow, progress bar
```

## Running it

No setup needed. Clone or download and open `index.html` directly.

```bash
git clone https://github.com/prabesha1/portfolio
cd portfolio
open index.html
```

Or just drag `index.html` into your browser.

## About the project

This portfolio was built as part of the COMP 3078 Capstone Portfolio deliverable. It covers:

- My background and career goals
- Work experience (Tim Hortons, freelance photography, Karma Events)
- Capstone project — [Shiftora](https://github.com/prabesha1/Shiftora), a full-stack employee scheduling app
- Academic work samples from GBC courses
- Certifications including First Aid, WHMIS, CRA CVITP volunteer recognition
